## 참고: 현재 구현된 단축키/기능

| 분류 | 키/동작 |
|---|---|
| 도구 | `V` 선택 · `P` 점 · `L` 선 · `H` 팬 |
| 편집 | `Delete`/`Backspace` 삭제 · `Esc` 취소·선택해제 · 방향키 넛지(선택 시, `Shift`=10배) · 선택 객체 마우스 드래그 이동(`Alt`=자유 이동, 기본 그리드 스냅) |
| 실행취소 | `Cmd/Ctrl + Z`(`+Shift` 재실행, `Ctrl+Y`) |
| 보기 | 휠 줌(커서 기준) · 키보드 줌(`+`/`-`, 뷰 중심) · 뷰 맞춤(`F`) · 방향키 팬(미선택 시, `Shift`=큰 보폭) · `Space` 누른 채 드래그 팬 · 휠(가운데) 버튼 드래그 팬 · Pan 도구(`H`) |
| 선택 | 클릭 단일 선택 · `Ctrl`/`Shift`+클릭 추가·해제(단일 종류) · 전체 선택(`Cmd/Ctrl + A`, 현재 마퀴 대상) · 드래그 마퀴(완전 포함만, `Shift`=선/`Ctrl`=면) |
| 입출력 | `Cmd/Ctrl + S` 내보내기(Export `.poly`) |
| HUD | 좌하단 커서 좌표(`x, z` m)·배율(`m/px`) 상시 표시 |
